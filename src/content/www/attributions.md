This site stands on a lot of other people's work. Here's what it's built with, and who to thank.

## Fonts

- **PP Agrandir** — display and headings. By [Pangram Pangram Foundry](https://pangrampangram.com/products/agrandir).
- **iA Writer Quattro** — body text. By [iA Inc.](https://github.com/iaolo/iA-Fonts), under the SIL Open Font License 1.1.
- **IBM Plex** — UI and labels. By [IBM](https://www.ibm.com/plex/), under the SIL Open Font License 1.1.
- **Lilex** — code and monospace. By [Misha Schwartz](https://github.com/mishamosher/Lilex), under the SIL Open Font License 1.1.
- **Zed Season** — accents. By [Zed Industries](https://zed.dev).

## Built with

- **[Pelican](https://getpelican.com/)** — the static site generator behind every page here. AGPL-3.0.
- **[Pygments](https://pygments.org/)** — syntax highlighting for the code blocks. BSD-2-Clause.
- **[MathJax](https://www.mathjax.org/)** — typesetting the math in a few posts. Apache-2.0.

- The look and feel of this theme is inspired by **[Zed](https://zed.dev)** by Zed Industries — the editor's site is a small masterpiece of restraint.
